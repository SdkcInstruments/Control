# Control
Controlは4つのOffset/Attenuverterからなる、1:4Demultiplexerです。

通常のCVインプットに加えて、Expressionペダルを使用することで、Eurorackの演奏に足でのコントロールという重要な要素を追加できます。  

各チャンネルのCVインプットに何も接続されていない場合、Master KnobとExpedal Inの入力が全てのチャンネルに内部接続されています。Ch2以降のCV入力にケーブルが接続されると、内部結線はそのポイントでカットされ、それ以降も同様に動作します。

例) Ch3の入力に外部からのCVを入力した場合、Ch1とCh2にはMaster KnobとEXPinのCVが接続され、Ch3とCh4にはCh3に入力したCVが接続されます。

# Control's controls
### Master Knob
0V-8VのCVを操作します。この入力は通常状態では全てのノブに内部結線されています。

### EXP in
ギターなどで使われる、Expressionペダルを接続する事ができます。0V-8Vを出力します。  
Controlでは一般的に用いられているTRS接続のコントロールペダルに対応しており、  

・CW-Sleeve / Wiper-Tip / CCW-Ring  
・CW-Sleeve / Wiper-Ring / CCW-Tip

裏面のスイッチで上記の極性を切り替える事ができます。


### Attenuverner  
入力された信号に対して、*-1~*1の増幅を行います。

### Offset
-8V~8Vのオフセット電圧を加えます。

OffsetとAttenuverterノブはセンタークリックがついていますが、アナログ回路を使用しているため、センターで必ずしも0になる訳ではありません。
